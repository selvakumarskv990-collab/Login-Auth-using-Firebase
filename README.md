#  Firebase Authentication (Login & Register)

A simple web app for beginners that shows how to use **Firebase Authentication** with **HTML, Bootstrap and JavaScript** — without any backend or database.

This project allows users to:
- Register using **email and password**
- Login securely
- Stay signed in using browser persistence
- Logout easily

---

##  Features

✅ Register new users using Firebase Authentication  
✅ Login with existing credentials  
✅ “Remember Me” option using local/session persistence  
✅ Toggle between login and register forms  
✅ Logout functionality  
✅ Clean and responsive UI (Bootstrap)  
✅ 100% frontend-only — no database required  

---

| Technology | Purpose |
|-------------|----------|
